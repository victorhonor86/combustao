Combustão Completa

Escolha entre digitar os valores dos átomos do combustível em "calculador" ou escolher entre os combustíveis para gerar a fórmula de combustão completa em "seletor".

Digite os valores para cada elemento químico:

C H O + testeO2
⇩
testeCO2 + testeH2O

Escolha um Combustível:
